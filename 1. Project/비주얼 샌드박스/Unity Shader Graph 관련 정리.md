## 스크롤 + Noise

> 2026-05-02

[Link](https://github.com/fkdl0048/VisualSandbox/tree/main/Assets/001_shaderGraph)

- 처음 만들어보는 쉐이더 그래프
- A, B는 계산 순서

### Add (더하기)

👉 값을 **그대로 더함**

```
결과 = A + B
```

- 위치 이동
- 밝기 증가
- 값 “밀어넣기”
### Multiply (곱하기)

👉 값을 **비율로 조절**

```
결과 = A × B
```

- 강도 조절
- 스케일 변경
- 영향력 조절

### 기본 UV

```
UV = (x, y)
```

👉 이미지 좌표

---

### 2. Time으로 흐름 만들기

```
Time × Vector2(0, 0.5)
```

👉 시간에 따라 계속 증가하는 값

```
(0, t * 0.5)
```

👉 위로 흐름

---

### 3. Offset 적용

```
UV + 시간값
```

👉 이미지가 계속 위로 이동

---

### 4. 노이즈 생성

```
Noise(UV)
```

👉 울퉁불퉁한 값 생성

---

### 5. 노이즈 강도 조절

```
Noise × 0.03
```

👉 흔들림을 작게 만듦

---

### 6. UV에 노이즈 추가

```
UV + Noise
```

👉 이미지가 울렁거리게 됨

---

### 7. 최종 UV로 텍스처 샘플

```
Texture(UV)
```

👉 결과 이미지 생성
